Title: \\
Long-term Memory Property for Fractional Brownian Fields : Cross Long-term Memory

Abstract: 
This chapter studies the long-term memory property for a fractional Brownian field. 
The fruit of this research is a novel concept: the cross long-term memory, which captures the long-term
memory of among cross variables in the random field. This new property allows to describe
specific financial phenomena that a single stochastic process is unable to catch. For example, if
a financial instrument is described by multiple factors such as interest rate and volatility, there
may exist non-ignorable correlation among the returns of the prices, the interest rates and the
volatilities. The cross long-term memory aims at parameterizing this feature. We will give a
mathematical definition of the cross long-term memory property and study its characteristics.
We will also perform a simulation study and a real world analysis, showing the existence of
such property and its interests in the quantitative financial study.
